# Retail Sales & Profitability Analysis using MySQL and Tableau

## Project Overview

This project analyzes retail sales performance using MySQL and Tableau.
The objective was to identify key revenue drivers, profitable product
categories, and high-performing regions through data analysis and
visualization.

## Business Problem

Retail businesses generate large volumes of transactional data. The goal
of this project was to analyze sales and profitability trends to support
data-driven decision-making and identify opportunities for improving
business performance.

## Dataset Information

-   Dataset: Sample Superstore
-   Records: 9,994 transactions
-   Fields Included:
    -   Category
    -   Sub-Category
    -   Region
    -   State
    -   Sales
    -   Quantity
    -   Discount
    -   Profit

## Tools Used

-   MySQL
-   Tableau Public
-   Microsoft Excel

## SQL Analysis Performed

Key SQL operations used: - Aggregate Functions (SUM, AVG) - GROUP BY -
ORDER BY - Filtering and Ranking - Business Performance Analysis

### Sample Business Questions

1.  What is the total sales revenue and profit?
2.  Which category generates the highest revenue?
3.  Which region contributes the most sales?
4.  Which states generate the highest revenue?
5.  How do product categories compare in profitability?

## Key Findings

-   Technology generated the highest sales revenue (\$836K).
-   Technology was also the most profitable category (\$145K profit).
-   West region recorded the highest revenue (\$725K) and profit.
-   California was the highest revenue-generating state (\$457K).
-   Office Supplies delivered stronger profitability than Furniture
    despite lower sales.

### Visualizations

## Business Recommendations

-   Increase focus on Technology products due to strong sales and
    profitability.
-   Prioritize high-performing markets such as California and New York.
-   Review pricing and discount strategies for Furniture products to
    improve margins.
-   Continue investing in the West region, which consistently delivers
    strong performance.

## Dashboard Preview

The Tableau dashboard provides an interactive overview of sales, profit,
quantity sold, category performance, regional performance, and

## Conclusion

This project demonstrates the use of SQL for business analysis and
Tableau for interactive data visualization. The analysis highlights key
revenue drivers and profitability trends that can support strategic
business decisions.
